# Primeiro Projeto em Godot

Meu primeiro projeto desenvolvido utilizando a Godot Engine.

Este projeto foi criado como parte dos meus estudos em desenvolvimento de jogos e teve como base um tutorial. Além de acompanhar o conteúdo, implementei algumas funcionalidades por conta própria para praticar os conceitos aprendidos.

## Tecnologias

- Godot 4
- GDScript

## Funcionalidades

- Menu inicial
- Navegação do menu por mouse
- Navegação por teclado
- Sons na interface
- Colisões
- Animações

## O que aprendi

Durante este projeto pratiquei:

- Estrutura de Nodes
- Scenes
- GDScript
- Signals
- Interface (UI)
- Áudio
- Física 2D
- Organização de projetos

## Modificações realizadas

Além do tutorial original, adicionei:

- Menu inicial
- Navegação utilizando teclado
- Sons ao selecionar opções do menu

## Como executar

1. Instale a Godot Engine 4.
2. Clone este repositório.
3. Abra o arquivo `project.godot`.
